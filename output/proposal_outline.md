# 해군교전분석모델 Build-II 결과분석 제안서 개요

## 제안 방향
- 기능 소개 중심이 아니라 RFP 요구사항 대응 중심으로 구성한다.
- 결과분석 데이터, 3D 가시화, 자동 재생, 확률/전자전 효과 분석, 시험평가 대응을 핵심 축으로 둔다.
- 확인 불가한 회사명, 실적, 인력, 인증은 기재하지 않고 역할·제안 방향으로 표기한다.

## Task 반영 요약
결과 분석
- 구조
1. 결과분석 데이터 3D 가시화 개선
2. 모의결과 데이터를 기반으로 자동 3D 재생
3. 멀티뷰 기능을 통해 전장 환경과 플랫폼 및 무기체계의 상호 작용을 전시
4. 고행상도 수중환경(수온, 분포, 음파전달손실음선경로) 3D 가시화
5. 계산모델 가시화
6. 탐지확률 명중확률등 수학적계산 결과를 실시간 차트로 표현
- 구성
1.유도탄, 기만기 기만 성공확률 산출 개선
2.어뢰, 기만기 기만 성공활률 산출 개선
3.전자전 장비(재머, GPS 교란 등)효과 산출 추가

## 장표 구성
- 총 26장 구성
- 한 장표당 Bullet 3~5개 기준
- RFP 원문 이미지와 PowerPoint 도형 기반 자동 도식을 병행 사용
